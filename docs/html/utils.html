utils.generate_images_for_genre(genre, num_samples=6, overlap=False)

Generate images from mel spectrogram segments for a given genre and save them to the same directory as the segments.

Parameters:

  • genre – string of genre to generate images for

  • num_samples – number of samples. Defaults to 6.

  • overlap – boolean determining whether slices of audio file will be overlapped or distinct. Defaults to False. Needed here for directory purposes.

Returns:

False if error, True if successful

utils.get_data_and_labels(overlap=False)

Extracts data and labels from the mel spectrogram images and returns them as numpy arrays.

Parameters:

overlap – boolean determining whether slices of audio file will be overlapped or distinct. Defaults to False. Needed here for directory purposes.

Returns:

data and labels as numpy arrays

utils.get_melspectrogram(wav_file_path, length=30, duration_of_segments=5, overlap=False, duration_of_step=1)

Get mel spectrogram for a given wav file and divide it into parts.

Parameters:

  • wav_file_path – Path to the source wav file

  • length – length in seconds of the source audio file. Defaults to 30.

  • duration_of_segments – duration of segments in seconds. number of segments = length/duration_of_segments. Defaults to 5.

  • overlap – boolean determining whether slices of audio file will be overlapped or distinct. Defaults to False

  • duration_of_step – step size from the beginning of one segment to the beginning of the next. Defaults to 1 second. Unused if overlap is False.

Returns:

Mel spectrogram of the source wav file. Segments will be saved to a file and it’s path will be printed.
